Why this trips people up

The bar itself already weighs 20kg (or 45lb), and plates get added to both sides — so per-side plate math is always half of the remaining weight, not the total.

The greedy algorithm

We fill each side using the largest plates first, then work down to smaller ones — the same way most lifters naturally load a bar in the gym.

When it won't land exactly

Not every weight is achievable with a standard plate set in fine 1.25kg or 2.5lb increments — if that happens, we show the closest achievable weight.

How it works

# Per side
Per side = (target weight - bar weight) / 2

# Standard plates available (kg)
25, 20, 15, 10, 5, 2.5, 1.25

# Standard plates available (lb)
45, 35, 25, 10, 5, 2.5

Largest plates are used first, working down to the smallest, on each side.
